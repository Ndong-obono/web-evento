# 💍 Proyecto: Web de Evento (Boda)

Una página web estática, responsiva y semántica diseñada para presentar los detalles de un evento especial, gestionar la información del lugar y facilitar la confirmación de asistencia.

## 🚀 Tecnologías Utilizadas

* **HTML5:** Estructura semántica y accesible.
* **CSS3:** Estilos, layout (Flexbox/Grid) y diseño responsivo.
* **Git & GitHub:** Control de versiones.

## 📁 Estructura del Proyecto

El proyecto sigue una arquitectura clásica de sitio estático para mantener la separación de responsabilidades:

- `/assets/img/`: Recursos gráficos e imágenes.
- `/css/`: Hojas de estilo en cascada.
- `/js/`: Scripts para interactividad (DOM).
- `index.html`: Punto de entrada de la aplicación.

## 🛠️ Instalación y Uso Local

Para correr este proyecto en tu entorno local, sigue estos pasos:

1. Clona el repositorio:
   \`\`\`bash
   git remote add origin https://github.com/Ndong-obono/web-evento.git
   \`\`\`
2. Abre la carpeta en tu editor de código preferido (ej. VS Code).
3. Abre el archivo `index.html` en tu navegador web o utiliza la extensión "Live Server" en VS Code para recarga en caliente.
